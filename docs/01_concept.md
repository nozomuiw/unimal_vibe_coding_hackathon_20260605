# 01. サービスコンセプト

## 解決したい課題

家庭内の小さな争い（家事分担、お金の使い方、生活習慣、子育て方針の食い違いなど）は、

- 当事者同士だと**感情が先に立って事実の整理ができない**
- 第三者に相談するほどではないが、**モヤモヤが蓄積する**
- 「どちらが正しいか」の決着がつかないまま、**同じ争いが繰り返される**

という構造を持つ。友人や親に相談しても中立ではなく、調停やカウンセリングは大げさすぎる。「ちょうどいい中立の第三者」が家庭内に存在しない。

## 提供価値

1. **中立性**: AIは双方のどちらにも利害関係がない。代理人AIは各人の側に立つが、裁判官AIは社会通念をベースに判断する
2. **言語化の支援**: 代理人AIのヒアリングを通じて、本人も整理できていなかった事実・感情・要望が言語化される
3. **エンタメ性による緩和**: 「裁判」というメタファーとキャラクター性のある裁判官により、深刻な対立を**ゲーム的な儀式**に変換する。判決に納得できなくても「あの裁判官だからな」と笑って受け流せる余白を作る
4. **決着がつく**: 曖昧なまま終わらせず、必ず「判決」というアウトプットで一区切りつける

## 設計上の重要な思想

### 裁判官の「ずれ」は意図的な仕様

裁判官はランダムに選定され、それぞれキャラクターを持つ。判定は社会通念上の公平性をベースにしつつ、**キャラクターによって公平性から少しずれる余白**を持たせる。

これは欠陥ではなく中核的な仕様である:

- 完全に公平な判定は、負けた側に逃げ場がない。「この裁判官は人情に厚いから」「あの裁判官は理屈っぽいから」という**キャラへの帰責**が、敗訴側の感情の受け皿になる
- どの裁判官に当たるかわからないランダム性が、結果への過度な期待を抑制し、ゲーム性を生む
- ただし、ずれは「味付け」の範囲に留める。社会通念から大きく逸脱した判決（一方的な断罪、危険な助言）は出さない。詳細は [03_characters.md](03_characters.md) のバイアス設計を参照

### 代理人は「味方」である

各代理人は基本的に該当人の弁護をする。ヒアリング時は共感的に話を引き出し、法廷では依頼人に有利な論を張る。**本人の言い分を本人以上にうまく言語化してくれる存在**であることが、このサービスへの信頼の土台になる。

「AIに話したら、自分の味方をせずに説教された」という体験は最悪のパターンであり、避ける。説諭は裁判官の役割であって代理人の役割ではない。

## 想定ユースケース（例）

- 「皿洗いの分担が不公平だ」
- 「相手が予算を相談せずに高い買い物をした」
- 「休日の過ごし方で毎回もめる」
- 「子どもへの叱り方が甘すぎる／厳しすぎる」
- 「義実家との付き合い方の温度差」

## 対象外領域（セーフティ境界）

以下はこのサービスの対象外であり、検知した場合は判決を出さず、適切な相談窓口の案内に切り替える:

| 領域 | 対応 |
|---|---|
| DV・身体的/精神的虐待の示唆 | 裁判ゲームを中断し、専門相談窓口（配偶者暴力相談支援センター等）を案内 |
| 児童虐待の示唆 | 同上（児童相談所等） |
| 離婚・財産分与など法的効力が必要な紛争 | 「本サービスは法的助言ではない」ことを明示し、弁護士・法テラス等を案内 |
| 自傷・他害の示唆 | ゲームを中断し、こころの健康相談等を案内 |

また、全フェーズで以下を明示する:

> 本サービスはエンターテインメントであり、法的助言・医療的助言・カウンセリングではありません。判決に法的効力はありません。

## プロトタイプのゴール

- AさんとBさんが1台の端末を交代しながら、**問題提起 → 聞き取り → 代理戦争 → 判決**の一連の体験を完走できる
- 裁判官のキャラクターの違いが体験として感じられる（同じ事案でも裁判官によって判決の味が変わる）
- 体験後に「面白かった」「整理された」のどちらか（できれば両方）の感想が得られる

プロトタイプで**やらないこと**: ユーザー登録・認証、課金、履歴の永続化、別端末からの非同期参加、多言語対応。
